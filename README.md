Project Description

Doctor Appointment System is a web-based application designed to simplify the process of booking medical appointments. The platform allows patients to register, view available doctors, and book appointments online. Doctors can manage their schedules, and the system helps reduce waiting time and manual appointment handling.

Features

User registration and login for patients and doctors
Doctor profile management
Online appointment booking system
View and manage appointment schedules
Simple and user-friendly interface
Secure handling of user data

Technologies Used

CSS
JavaScript

Backend

Express.js

Database
MongoDB

Project Structure

server.js is the main backend entry point
package.json contains backend dependencies and scripts
config folder contains database connection files
controllers folder contains business logic for users and appointments
models folder contains database schemas
routes folder contains API routing
client folder contains the frontend application

How to Run the Project

Step 1 Extract the ZIP file

Step 2 Open the project folder in a code editor

Step 3 Install backend dependencies using
npm install

Step 4 Create a .env file and configure database details

Step 5 Start the backend server using
npm start

Step 6 Open the frontend application in a browser
[http://localhost:3000](http://localhost:3000)

Backend server runs on
[http://localhost:5000](http://localhost:5000)
